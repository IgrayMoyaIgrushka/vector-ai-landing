# История изменений проекта Vector AI Landing

## 2026-03-11 — Улучшение дизайна, анимаций и мобильной версии

### Что сделано:
- **Добавлена кнопка «Наверх» (Scroll to Top):**
  - Появляется при прокрутке более 500px
  - Плавная анимация появления/исчезновения
  - Золотой градиент в стиле лендинга
  - Анимация стрелки при наведении

- **Улучшена мобильная версия:**
  - Добавлено гамбургер-меню с анимацией превращения в крестик
  - Выезжающее справа меню с оверлеем
  - Закрытие по клику на оверлей, кнопку закрытия или Escape
  - Адаптивное меню содержит все навигационные ссылки
  - Переключатель языков RU/EN в мобильном меню
  - Кнопка Telegram в мобильном меню

- **Добавлены новые анимации:**
  - **Параллакс-эффект** для фоновых элементов (орбы, частицы, сетка)
  - **Плавающие иконки** (case-card-icon, meta-icon)
  - ** Shine-эффект** при наведении на карточки (пробегающая полоса света)
  - **Pulse-анимация** для CTA-кнопок (пульсирующая тень)
  - **Ripple-эффект** при клике на кнопки
  - **Анимация границы** для hero-card и contact (переливающаяся граница)
  - **Glow-эффект** для цифр шагов (размытое свечение)
  - **Улучшенный hover** для карточек с bounce-эффектом
  - **Интерактивный glow** следует за курсором мыши

- **Улучшены существующие анимации:**
  - Увеличена амплитуда появления секций (50px вместо 30px)
  - Использован IntersectionObserver для производительности
  - Добавлена плавная прокрутка для якорных ссылок
  - Улучшены transition для карточек (cubic-bezier)

- **Доступность:**
  - Добавлены focus-visible состояния для клавиатурной навигации
  - Поддержка prefers-reduced-motion для пользователей с вестибулярными расстройствами
  - ARIA-лейблы для кнопок

### Файлы:
- `index.html`: +126 строк (мобильное меню, кнопка наверх)
- `style.css`: +510 строк (стили мобильного меню, новые анимации)

### Технические детали:
- IntersectionObserver для reveal-анимаций (лучшая производительность)
- CSS custom properties для консистивности
- will-change для оптимизации анимаций
- Блокировка прокрутки body при открытом мобильном меню

### Что сделано:

### Структура проекта:
```
landings/
├── index.html      (HTML структура, 681 строка)
├── style.css       (все стили, 1814 строк)
├── server.js       (сервер для Telegram форм)
├── package.json
├── .env            (переменные окружения, не коммитить!)
└── render.yaml     (конфигурация Render)
```
